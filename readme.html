This is a simple example:

  • It's a standard html file (check readme.html)
  • It uses some formatting (check the file styles.css)
  • It uses a subtle background (noisy.png - found on subtlepatterns.com)
  • You can include PNG images (like icons or screenshots)
  • The html can contain links to websites or email-addresses that will be opened in Safari or Mail.

and ask you to check out the product's website or to send me an email.
